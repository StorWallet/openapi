# openapi

Run your own RPC node for [StorWallet](https://storwallet.com/): Web Wallet for Stor Blockchain

# Installation

```
git clone https://github.com/StorWallet/openapi.git

pip install -r requirements.txt

uvicorn openapi:app
```
